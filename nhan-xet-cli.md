# Nhan xet ca nhan ve Copilot CLI va Gemini CLI

## Cong cu huu ich hon
Trong bai thuc hanh nay, Copilot CLI huu ich hon khi lam viec voi GitHub workflow vi co muc tich hop voi he sinh thai GitHub.

## Ly do
- Copilot CLI de lap ke hoach tac vu code va thao tac lien quan repository.
- Gemini CLI manh o kha nang mo rong va tham chieu file linh hoat bang cu phap `@file`.
- Neu can workflow gan voi GitHub thi Copilot CLI tien loi hon.
- Neu can trao doi tong quat va thu nghiem agent skills thi Gemini CLI linh hoat hon.

## Ket luan
Ca hai deu huu ich. Nen chon theo boi canh cong viec thay vi chi dung mot cong cu co dinh.
